# Problema proposto

Suponha que você esteja trabalhando em um sistema de gerenciamento de estoque para uma loja de roupas. O sistema atual foi desenvolvido há muitos anos e utiliza uma classe chamada EstoqueAntigo para controlar os produtos em estoque. Essa classe possui uma interface incompatível com a nova API de um fornecedor de roupas online, chamada EstoqueFornecedor.
